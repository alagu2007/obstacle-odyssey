# 🎮 Omit the Obstacles If You Can

> A custom 2D platformer game built from scratch using Python and the Pygame library. Originally developed as a comprehensive Non-Exam Assessment (NEA) project.

## 🚀 Overview
"Omit the Obstacles If You Can" is a fully realized platformer that tests player reflexes with dynamic gameplay elements. It features a complete game loop, robust collision mechanics, and is structured entirely around Object-Oriented Programming (OOP) principles.

## ✨ Key Features
* **Custom Physics Engine:** Engineered from scratch in Python to handle gravity, jumping, and precise player movement.
* **Dynamic Gameplay:** Features randomized obstacles and dynamic enemy behaviors to keep the levels engaging and challenging.
* **OOP Architecture:** Built using modular classes for scalable game state management, hitboxes, and sprite rendering.
